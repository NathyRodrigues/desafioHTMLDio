O que é HTML?

[voltar]

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de marcação utilizada na maioria das páginas da internet. Sendo uma linguagem de marcação, o HTML é composto por marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento. O HTML não é considerado uma linguagem de programação pois ele não pode criar funcionalidades dinâmicas. Com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

História do HTML

[voltar]

O HTML foi inventado por Tim Berners-Lee na Suíça. Ele surgiu com a ideia de um sistema de hipertexto na internet. Hipertexto significa um texto que possui referências (links) para outros textos que podem ser acessados imediatamente. Ele publicou a primeira versão do HTML em 1991, com 18 tags. Desde então, cada versão do HTML vem com novas tags e atributos, que são os modificadores das tags. Por sua alta popularidade e rápida ascensão, o HTML é agora considerado um padrão oficial da web.

Tags HTML

[voltar]

As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere. Atualmente, existem 142 tags HTML disponíveis, que permitem a criação de diversos elementos. Aqui apresentaremos as tags HTML mais usadas na estrutura básica do HTML.

A estrutura básica

O VS Code, que é uma IDE (ambiente de desenvolvimento integrado) utiliza por padrão o Emmet Abbreviations, que traz o aparecimento automático de linhas de código que fazem parte da estrutura básica do HTML ao digitar o ponto de exclamação. Assim:

!DOCTYPE html

Informa ao navegador que esse documento é do tipo HTML e sua versão.

HTML

Representa a raiz do documento, serve com um container que engloba todos os outros elementos HTML.

Body

É onde fica todo o conteúdo de texto, imagem e vídeos, em que o usuário vê e interage, nele os conteúdos são estruturados pelas demais tags do HTML.

Head

Compreende as informações do documento que serão interpretadas pelo navegador (metadados). Como por exemplo, título do documento, links para folhas de estilo etc.

Tags Semânticas

[voltar]

Tags semânticas são tags que possuem um significado, ou seja, dão sentido a informação de texto ao navegador e buscadores, como por exemplo, utilizar a tag header para cabeçalhos ou footer para dar significado de rodapé para aquele bloco de texto, até mesmo "p" para indicar que aquele texto é um parágrafo, é uma boa prática tentar sempre utilizar essas tags semânticas para ajudar no entendimento do código.

Exemplos de Tags Semânticas

  1. article
  2. aside
  3. footer
  4. header
  5. main
  6. nav
  7. entre outras...

Elementos como header, nav, section, article, aside e footer funcionam mais ou menos como elementos "div". Eles agrupam outros elementos juntos em seções de uma página. Uma tag "div" poderia conter qualquer tipo de informação. Contudo, é fácil identificar que tipo de informação seria incluída em uma tag semântica como "header". Curiosidade sobre Listas em HTML

[voltar]
Podemos criar listas em HTML para atender as mais diversas necessidades em uma página. O HTML tem algumas tags definidas para que possamos fazer isso de maneira correta. A lista mais comum é a não-ordenada definida pela tag ul - (unordered list). As listas ordenadas (ol - ordered list) também podem ter sua estrutura composta por outras listas ordenadas como no exemplo que temos para as listas não-ordenadas. Também é possível ter listas ordenadas aninhadas em um item de uma lista não-ordenada e vice-versa.
